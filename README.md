# Ping pong game TronLegacy version

## How to configure

```bash
./configure.sh
```

</br>


## How to run

```bash
PLAYER_1_NAME=Dad PLAYER_2_NAME=Mom ./run.sh
```

</br>

If you don't provide the above env vars, it uses the default settings:

```bash
PLAYER_1_NAME=Player 1
PLAYER_2_NAME=Player 2
```

</br>
